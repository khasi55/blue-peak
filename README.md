# Jaspro Solutions - React Application

This is a modern React application built with Vite and TypeScript for Jaspro Solutions.

## Prerequisites
*   Node.js (18+)
*   npm (9+)

## Project Structure
*   `src/pages` - Application pages (Home, About, Services, etc.)
*   `src/components` - Reusable components (Navbar, Modal, Footer)
*   `src/index.css` - Global styles and utility classes
*   `src/App.tsx` - Main routing configuration

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
